Project Overview Summary

CartNexus is a full‑stack bilingual (English / Bangla) web store for men’s fashion: browsing, cart, checkout, customer accounts, and an admin panel for catalog, orders, inventory, and content.

Goals
  • Modern storefront UX
  • REST API + MySQL persistence
  • Admin dashboard & inventory
Status

Core MVP implemented — testing & deployment in progress.

Indicative completion (~78%)

Challenges Mitigation

  • Schema drift: migrations + changelog discipline

  • Payments: COD baseline · gateway integration = future work
